# Setup Project

```
npx @muhammadisa226/create-express-ts-prisma-starter-kit my-app
```

```shell
npm install
```

```shell
npm run generate
```

```shell
npx run migate:run
```

```shell
npx run migrate:seed
```

# Run Development

```shell
npm run dev
```

# Build To Production

```shell
npm run build
```

```shell
npm start
```
