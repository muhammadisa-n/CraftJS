# Setup Project

```
npx @muhammadisa226/create-express-ts-prisma-starter-kit my-app
```

```shell
npm install
```

```shell
npx prisma generate
```

```shell
npx prisma migrate dev
```

# Run Development

```shell
npm run dev
```

# Build To Production

```shell
npm run build
```

```shell
npm start
```
