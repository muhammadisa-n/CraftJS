# Install Project

```shell
npx @muhammadisa226/create-express-ts-prisma-starter-kit@latest my-app
```

# Setup Project

```shell
npm run install
npm run craft key:generate
npm run craft db:generate
npm run craft db:migrate
```

# Run Development

```shell
npm run craft dev
```

# Build To Production

```shell
npm run craft build
```

```shell
npm run craft start
```

# Show All Command Craft

```shell
npm run craft help
```
