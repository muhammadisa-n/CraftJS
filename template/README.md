# Express Prisma Typescript Starter Kit

# Setup Project

```
DATABASE_URL="mysql://username:password@host:port/databasename"
```

```shell
npm install
```

```shell
npx prisma generate
```

```shell
npx prisma migrate dev
```

# Run Development

```shell
npm run dev
```

# Build To Production

```shell
npm run build
```

```shell
npm start
```
